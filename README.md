# Formula-1-Performance-Analytics
Build a performance analytics platform to evaluate driver consistency, team strategy efficiency, and race outcomes using Formula 1 data.
Seasons: 2023, 2024, 2025
Data source: FastF1
Data extraction and cleaning : python
Data modelling : Power BI
Output: Power BI dashboard
Analysis focus:
    Driver performance
    Team strategy (pit stops, tyres)
    Race outcomes
